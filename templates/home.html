Django Rest Framework Assignment
1. Register an employee ( By Admin ) ( POST )
2. Login as employee ( POST )
3. Update employee details ( by employee ) ( PUT/PATCH )
4. Get paginated list of employees ( Ascending order by creation date ) ( GET )

Solution -
1. To register an employe(only Admin can register new employees) -
        a. Login in as admin - http://pragatitask.pythonanywhere.com/api/login/
        b. Go to - http://pragatitask.pythonanywhere.com/api/department/register
        c. Post data in this format -
{
"username" : "WIZ04Ritu",
"password" : "User1234",
"email" : "ritu@gmail.com",
"name" :"Ritu Kapoor",
"dept": 1

"join": "2021-05-18",
"dob": "1998-12-25",
"phone": "1234554321",
"address": "Adarsh Nagar, Jaipur",
}
(Last 4 are optional)

2. Employee can log in using - http://pragatitask.pythonanywhere.com/api/login/

3. Employee can update (only) their details by using - http://pragatitask.pythonanywhere.com/api/employee/update
eg - {"phone":"8945735350"} will update phone number.

4. View Complete Employee list at (Pagination-5) - http://pragatitask.pythonanywhere.com/api/employee/list

5. Admin can add/view department by using - http://pragatitask.pythonanywhere.com/api/department/list/
